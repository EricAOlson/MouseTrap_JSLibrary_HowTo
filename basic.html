BASIC KEY BINDING

.bind() Method

The Mousetrap object contains an associative array of all the key bindings it needs to monitor for. Initially, this array is empty, so to begin using Mousetrap we need to bind keys:events to the array. This is done with the .bind() method. This method is the workhorse of the Mousetrap library, and much of this guide is dedicated to various ways to leverage this single method.

Mousetrap.bind('key',  function(),  type);

The method takes 3 arguments:

  • 'key' - The key(s) to be bound, and the options available here will be covered in more detail in this and the next sections of the How-To.
  • function() - A basic JavaScript function call that executes the action you wish to perform when the 'key' binding is triggered. This can be defined inline in the .bind() command or separately (var assigned a function). Essentially, anything you can do within a JavaScript function can therefore be bound to a key-binding.
  • type - This is an optional argument that can accept one of three values:
    • 'keypress'
    • 'keydown'
    • 'keyup'
Mousetrap will make an informed decision as to the intended type based on the bound keys. These should only be used if you have a very specific reason to overwrite the default functionality.

Supported Keys

Most standard keys can be bound via Mousetrap.bind(), these include:

  • Alpha (a, b, ... , z)
  • Numeric (1, 2, ... , 0)
  • Upper Case* (A, B, ... , Z)
  • Special Characters* (~, !, @, ... , -, =)
  • Specialty Keys, these include keys that can not easily be typed directly into JavaScript. Instead you type the keyword in place of the key itself:
    • 'alt'
    • 'backspace'
    • 'capslock'
    • 'command' (for Mac)
    • 'ctrl'
    • 'del'
    • 'down' (down arrow)
    • 'end'
    • 'enter'
    • 'esc' or 'escape'
    • 'home'
    • 'ins'
    • 'left' (left arrow)
    • 'meta'
    • 'option' (for Mac)
    • 'pagedown'
    • 'pageup'
    • 'plus'
    • 'return'
    • 'right' (right arrow)
    • 'shift'
    • 'space'
    • 'tab'
    • 'up' (up arrow)
* Note: Capital Letters and symbols that require a shift to access (i.e. !, @, #, etc.) are technically key combos. While combos are also available and covered in the next section, Mousetrap interprets capital letters and shift requiring symbols automatically. This means mousetrap.bind('A', varFunction) or mousetrap.bind('%', varFunction) are perfectly legal.

Example

Let's say you're hosting a nerd website with jokes just for nerds. You want to present the user with the setup, and then 'reveal' the punch line to the user using key bindings. This simple example can be implemented with the .bind() method and CCS styling.

Mousetrap.bind('a', function() {
  var ans = document.getElementById('answer');
  ans.className = 'show';
  }
);
