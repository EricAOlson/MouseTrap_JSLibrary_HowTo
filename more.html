ADDITIONAL TOOLS

While this guide provides an overview of the most basic functionality of the MouseTrap library, there is much more available to users. If you are interested in additional functionality, please explore the provided documentation with the library at the Github Repo or checkout some of the other 3rd Party extensions available, such as: mousetrap-rails, which integrates Mousetrap with Rails.

Some additional functionality worth exploring:

  • Wrapping only specific document elements with a key binding.
  • Overwriting default browser key-bindings such as 'ctrl-n' for new tab.
  • Additional Mousetrap Extensions including:
    • Bind Dictionary - Allows you to consolidate multiple .bind requests into a single command.
    • Record - Allows users to record their own key binding sequences and combinations.
    • Trigger - Allows you to manually trigger key bound events without user input.

I would also like to point out another JavaScript library I discovered while researching Mousetrap. It's the rainbow library, created by the same author of Mousetrap. The library allows users to add CSS styling mirroring a variety of text editor like color schemes for various coding languages. I used the library, with modifications, on this site for any html/javascript examples.
