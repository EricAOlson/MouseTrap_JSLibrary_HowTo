LIBRARY SETUP

Get Mousetrap Library

The library is a small javascript file that can be downloaded as part of ccampbell's mousetrap repository or Mr. Campbell has also provided the following link, which you can "Save link as..." HERE. You're going to need 'mousetrap.min.js', a small 4.57KB file.

The above links provide the minimized library. If you would like to review the library in a more understandable user friendly layout (i.e. with white space and comments), download the mousetrap.js file instead from the same Github repo or HERE.

Include Mousetrap Library

To use the library, you simply include the file as a <script> in any HTML page. See below:

<script src="\PATH_TO\mousetrap.min.js"></script>
or
<script src="https://craig.global.ssl.fastly.net/js/mousetrap/mousetrap.min.js"></script>
Note the second option benefits from receiving updates to the library Mr. Campbell makes over time, but is conversely subject to Mr. Campbell's site remaining updated, and available, so use at your own risk.

Now you are ready to use the Mousetrap Library. An object called Mousetrap will be included in your JavaScript code. Mousetrap objects include a number of internal methods which will be covered throughout this How-To.
